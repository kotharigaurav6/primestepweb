Different ways to define a color in css : ----------------------------------------- 1) By Color Name EXAMLE : color:"red" 2) By Hexadecimal EXAMPLE : color:"#000000" = black color:"#ffffff" = white |-- # ff ff ff # f f f (shorthand way to define color) |-- # 44 22 55 # 4 2 5 (shorthand way to define color) |-- #345678 color:"#345643" 3) By RGB (R = RED, G = GREEN, B = BLUE) EXAMPLE : rgb(0,0,0) = black rgb(255,255,255) = white rgb(34,87,135) 4) By RGBA (R = RED, G = GREEN, B = BLUE, A = ALPHA) EXAMPLE : rgb(40,50,60,0.6) 5) By HSL (H = HUE, S = SATURATION, L = LIGHTNESS) EXAMPLE : hsl(60,50%,50%) 6) By HSLA (H = HUE, S = SATURATION, L = LIGHTNESS, A = ALPHA) EXAMPLE : hsl(60,50%,50%,0.3)
